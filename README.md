# 🎓 LearnFrame - Learn to Earn Platform on Base

Learn-to-Earn educational platform built on Base blockchain.

## Features
- 🎯 Interactive quiz system with token rewards
- 💰 ERC-20 LEARN token
- 🏆 Achievement NFTs
- 🔗 Base L2 blockchain integration
- 🖼️ Farcaster Frames support

## Smart Contracts
- **LearnToken.sol**: ERC-20 token for rewards
- **QuizManager.sol**: Quiz logic and reward distribution

## Tech Stack
- Blockchain: Base L2
- Smart Contracts: Solidity, Hardhat
- Frontend: Next.js, TypeScript
